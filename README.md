# Escola Sustentável do Futuro

Aplicação interativa desenvolvida em Python e Streamlit para apoiar análises pedagógicas, elétricas, civis e financeiras no planejamento de uma escola pública sustentável. A ferramenta permite ajustar parâmetros do projeto e visualizar indicadores em tempo real, facilitando discussões interdisciplinares durante as aulas de Pensamento Computacional.

## Instalação e Execução Local

Clone o repositório:

```bash
git clone https://github.com/0x03c1/streamlit-demo.git
cd streamlit-demo
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute a aplicação:

```bash
streamlit run app.py
```

---

## Funcionalidades

1. **Análise Pedagógica**

   * Cálculo de áreas para salas, horta, quadra e circulação.
   * Verificação da viabilidade pedagógica.

2. **Análise Elétrica**

   * Estimativa de consumo energético.
   * Cálculo de potência fotovoltaica necessária e custo do sistema.

3. **Análise Civil**

   * Estimativa de concreto, aço e custo da estrutura.

4. **Análise Financeira**

   * Simulação do orçamento total do projeto.
   * Cálculo de desembolso mensal e verificação de viabilidade financeira.

5. **Aprovação Geral**

   * O sistema informa se o projeto atende aos critérios mínimos para ser aprovado.

---

## Objetivo Educacional

A aplicação foi criada para integrar conceitos de Pensamento Computacional às áreas de pedagogia, engenharia civil, engenharia elétrica, administração pública e tecnologia. O objetivo é demonstrar como decisões quantitativas influenciam a viabilidade de um projeto real.
